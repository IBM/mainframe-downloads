IBM Record
Generator
for Java

IBM® Record Generator for Java™ helps Java developers write applications that interact with COBOL and Assembler applications on z/OS.

Download now Find out more

How does it work?

IBM Record Generator for Java is a stand-alone utility that imports the generated associated-data (ADATA) that is produced from compiling COBOL copybooks or Assembler DSECTs, and generates Java helper classes. These Java helper classes can then be used in a Java applications to marshal data to and from the COBOL or assembler language-specific record structures.

Much simpler to develop Java applications that have to interact with structured enterprise data.

More flexibility than ever before in the data that can be used as input and the format of the output.

Fully supported by IBM for fixes to issues or requests for enhancement – and free of charge.

IBM Record Generator for Java Version 3 supersedes the IBM alphaWorks version of the JZOS Record Generator V2.4.6 and provides a number of new capabilities
